### Setup Instructions

To run these tests, install test frameworks in each directory:

**Backend:**
```bash
cd backend
npm install --save-dev vitest @vitest/ui
npx vitest
```

**Frontend:**
```bash
cd frontend
npm install --save-dev vitest @testing-library/react @testing-library/dom
npx vitest
```
